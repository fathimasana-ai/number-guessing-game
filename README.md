# 🎮 Number Guessing Game

## Description
A simple Python game where the computer randomly selects a number between 1 and 10, and the user keeps guessing until the correct number is found.

## Features
- Random number generation
- User input
- While loop
- If-else statements
- Too high / Too low hints

## Skills Used
- Python
- Variables
- Input
- Random Module
- While Loop
- If-Else Statements
